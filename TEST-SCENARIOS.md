## ✅ 테스트 체크리스트

### 📝 1. 북마크 추가 기능

#### 시나리오 1-1: 모든 필드 입력하여 추가
- **준비 상태**: 브라우저에서 index.html 열기, LocalStorage 비어있음
- **입력/행동**:
  1. 제목: "React 공식 문서"
  2. URL: "https://react.dev"
  3. 카테고리: "개발"
  4. 설명: "React 학습용 공식 문서"
  5. "북마크 추가" 버튼 클릭
- **기대 결과**:
  - [ ] "북마크가 추가되었습니다!" 알림 표시
  - [ ] 폼이 초기화됨 (모든 입력 필드 비어있음)
  - [ ] 북마크 목록에 새 카드 표시
  - [ ] 카드에 제목, URL, 카테고리, 설명, 생성일 표시
  - [ ] 북마크 개수 "총 1개"로 업데이트
  - [ ] 빈 상태 메시지 사라짐

#### 시나리오 1-2: 선택 필드(설명) 없이 추가
- **준비 상태**: 북마크 1개 존재
- **입력/행동**:
  1. 제목: "MDN Web Docs"
  2. URL: "https://developer.mozilla.org"
  3. 카테고리: "개발"
  4. 설명: (빈 칸)
  5. "북마크 추가" 버튼 클릭
- **기대 결과**:
  - [ ] 정상적으로 추가됨
  - [ ] 카드에 설명 부분 없음
  - [ ] 북마크 개수 "총 2개"

#### 시나리오 1-3: 필수 필드 누락 시 검증
- **준비 상태**: 북마크 2개 존재
- **입력/행동**:
  1. 제목: (빈 칸)
  2. URL: "https://example.com"
  3. 카테고리: "개발"
  4. "북마크 추가" 버튼 클릭
- **기대 결과**:
  - [ ] "제목, URL, 카테고리는 필수 입력 항목입니다." 알림 표시
  - [ ] 북마크가 추가되지 않음
  - [ ] 북마크 개수 변화 없음

### 📤 9. Import 기능

#### 시나리오 9-1: 빈 상태에서 Import
- **준비 상태**: 북마크 0개, 사전에 Export한 JSON 파일 준비 (3개 북마크)
- **입력/행동**:
  1. "Import" 버튼 클릭
  2. JSON 파일 선택
- **기대 결과**:
  - [ ] "3개의 북마크를 가져왔습니다! (중복 0개 제외)" 알림 표시
  - [ ] 3개 북마크 카드 표시
  - [ ] 북마크 개수 "총 3개"
  - [ ] 즐겨찾기 상태 유지

#### 시나리오 9-2: 기존 데이터와 병합 (중복 없음)
- **준비 상태**: 북마크 2개 존재, Import용 JSON 파일 (3개 북마크, 모두 다른 ID)
- **입력/행동**:
  1. "Import" 버튼 클릭
  2. JSON 파일 선택
  3. 확인 다이얼로그에서 "확인" 클릭
- **기대 결과**:
  - [ ] 확인 메시지: "현재 2개의 북마크가 있습니다. ... 계속하시겠습니까?"
  - [ ] "3개의 북마크를 가져왔습니다! (중복 0개 제외)" 알림
  - [ ] 총 5개 북마크 표시 (2 + 3)

#### 시나리오 9-3: 중복 ID 체크
- **준비 상태**: 북마크 2개 존재 (ID: "1", "2")
- **입력/행동**:
  1. Export하여 JSON 파일 생성 (같은 ID: "1", "2" 포함)
  2. "Import" 버튼으로 같은 파일 가져오기
