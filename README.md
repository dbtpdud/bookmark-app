# 📚 북마크 관리 웹 앱

HTML, CSS, JavaScript로 만든 간단하고 강력한 북마크 관리 애플리케이션입니다.

## ✨ 주요 기능

### 기본 기능
- ✅ **북마크 추가**: 제목, URL, 카테고리, 설명을 입력하여 북마크 저장
- ✅ **북마크 목록**: 카드 형식의 깔끔한 그리드 레이아웃
- ✅ **북마크 수정**: 저장된 북마크의 정보 수정
- ✅ **북마크 삭제**: 확인 메시지 후 삭제

### 검색 및 필터링
- 🔍 **실시간 검색**: 제목, URL, 설명에서 키워드 검색 (대소문자 무시)
- 🏷️ **카테고리 필터**: 개발, 디자인, 뉴스, 유틸리티, 기타 카테고리별 필터링
- ⭐ **즐겨찾기 필터**: 즐겨찾기한 북마크만 모아보기

### 정렬 기능
- 📅 **최신순/오래된순**: 생성 날짜 기준
- 🔤 **제목순**: 가나다순 정렬
- 📁 **카테고리순**: 카테고리별 그룹화

### 추가 기능
- ⭐ **즐겨찾기**: 별표(⭐/☆) 토글로 중요한 북마크 표시
- 📥 **Export**: 모든 북마크를 JSON 파일로 다운로드
- 📤 **Import**: JSON 파일 업로드하여 북마크 복원 (중복 자동 체크)
- 💾 **LocalStorage**: 브라우저 종료 후에도 데이터 영구 보존
- 📱 **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원

## 🚀 사용 방법

### 1. 로컬에서 실행

```bash
# 저장소 클론
cd bookmark-app

# index.html 파일을 브라우저에서 열기
# 또는 Live Server 확장 사용

```

👉 [앱 바로 가기]

배포 주소:
https://dbtpdud.github.io/bookmark-app/

### 2. 북마크 추가하기
1. 상단 폼에 제목, URL, 카테고리 입력
2. 선택적으로 설명 추가
3. "북마크 추가" 버튼 클릭

### 3. 검색 및 필터링
- **검색**: 검색창에 키워드 입력 (실시간 필터링)
- **카테고리**: 드롭다운에서 원하는 카테고리 선택
- **즐겨찾기**: 별표 버튼 클릭 후 필터에서 "즐겨찾기만" 선택

### 4. 데이터 백업 및 복원
- **백업**: Export 버튼 클릭 → JSON 파일 다운로드
- **복원**: Import 버튼 클릭 → JSON 파일 선택

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 반응형 디자인
- **JavaScript (ES6+)**: Vanilla JS (라이브러리 없음)
- **LocalStorage API**: 클라이언트 사이드 데이터 저장

## 📊 데이터 구조

각 북마크는 다음과 같은 형식으로 저장됩니다:

```javascript
{
  id: "1638123456789",              // 고유 ID (timestamp)
  title: "React 공식 문서",          // 제목
  url: "https://react.dev",         // URL
  category: "개발",                  // 카테고리
  description: "React 학습용",       // 설명 (선택)
  createdAt: "2025-11-15T10:30:00", // 생성 날짜
  updatedAt: "2025-11-15T10:30:00", // 수정 날짜
  isFavorite: false                  // 즐겨찾기 여부
}
```
